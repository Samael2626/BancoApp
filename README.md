# BancoApp

Aplicación web Blazor para gestión de banca digital.

## Descripción
BancoApp es un proyecto de ejemplo construido con **ASP.NET Core Blazor**. Está diseñado para demostrar una aplicación web moderna con componentes de cliente y servidor.

## Requisitos
- .NET 8 SDK o superior
- Visual Studio 2022/2023 con soporte para Blazor

## Instalación
1. Clona el repositorio:
   ```powershell
   git clone https://github.com/Samael2626/BancoApp.git
   cd BancoApp
   ```
2. Restaura paquetes y ejecuta:
   ```powershell
   dotnet restore
   dotnet run --project BancoApp.csproj
   ```
3. Abre el navegador en la URL mostrada en la salida, típicamente `https://localhost:7098`.

## Estructura del proyecto
- `BancoApp.sln`: solución principal
- `BancoApp.csproj`: proyecto Blazor
- `Program.cs`: configuración de arranque
- `App.razor`: componente raíz
- `_Imports.razor`: directivas comunes

## Uso
- Ejecuta la aplicación con `dotnet run`.
- Modifica rutas, componentes y estilos según tus necesidades.
